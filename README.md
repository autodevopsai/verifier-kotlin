# Verifier (Kotlin)

A Kotlin/Gradle implementation of the AutoDevOps Verifier CLI.

- Language: Kotlin (JVM)
- Build: Gradle Wrapper
- Entry point: `com.autodevops.verifier.AppKt`

## Quick Start

- Build: `./gradlew build`
- Run: `./gradlew run --args="--help"`

## Requirements

- JDK 21+
